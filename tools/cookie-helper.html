🔐 OpenClaw Vault — Cookie Export

Export browser cookies in Playwright-compatible JSON for use with openclaw vault add

How to use:
1 Log into the target site in this browser
2 Open this page in a new tab (same browser)
3 Enter the target domain below
4 Click "Read Cookies" to capture accessible cookies
5 Copy the JSON output and paste into openclaw vault add <name> --type browser-cookie --domain <domain>
⚠️ Limitation: document.cookie cannot read httpOnly cookies. For full cookie capture, use browser DevTools → Application → Cookies, or a browser extension that can export all cookies. This helper captures what's available via the DOM API.
